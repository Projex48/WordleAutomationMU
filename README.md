# Wordle Automation
Express.JS RESTful API with automation to play a game of Wordle!

### Dependencies
  SeleniumWebDriver
  ExpressJS
  NodeJS

## Installation
  Download NodeJS from https://nodejs.org/en/download
  Open a terminal at this project directory
  Enter command: "npm install"

### Start Server
  Open a terminal at this project directory
  Enter command: "node server.cjs"

### Stop Server
  The server can be stopped at any time by exiting the terminal or with a keyboard interrupt (ctrl + c)
